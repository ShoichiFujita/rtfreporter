# Assemble multiple RTF files into one

Concatenates two or more RTF files generated by
[`generate_rtfreport`](https://ichirio.github.io/rtfreporter/reference/generate_rtfreport.md)
into a single RTF document. The document header (font table, colour
table, page settings) is taken from the **first** input file; subsequent
files contribute only their section content, separated by `\sect`.

This is the standard way to produce a full TFL package (T14.1.1,
T14.2.1, …) as one deliverable RTF.

## Usage

``` r
assemble_rtf(input_files, output_file, overwrite = FALSE)
```

## Arguments

- input_files:

  Character vector of paths to RTF files to combine. Must contain at
  least 2 elements. Files are included in order.

- output_file:

  Path for the assembled output RTF file.

- overwrite:

  Logical; whether to overwrite an existing output file. Default
  `FALSE`.

## Value

Invisibly returns `output_file`.

## Page numbering

Page-number fields work correctly across the assembled document when the
header/footer text uses *dynamic* tokens:

- `{AUTO_PAGE}` — per-page number rendered by the RTF viewer.

- `{AUTO_TOTAL_PAGES}` — total-page NUMPAGES field rendered by the RTF
  viewer across all assembled files.

The static token `{TOTAL_PAGES}` reflects only the page count of each
source file and will show the wrong total in the assembled output.

## See also

[`generate_rtfreport`](https://ichirio.github.io/rtfreporter/reference/generate_rtfreport.md),
[`rtf_document`](https://ichirio.github.io/rtfreporter/reference/rtf_document.md)

## Examples

``` r
if (FALSE) { # \dontrun{
  library(magrittr)

  # Build two reports and assemble them.
  make_report <- function(df, hdr) {
    rtf_document() %>%
      rtf_section(page = 1, secinfo = list(header = rtf_header(c(l = hdr)))) %>%
      rtf_tables(list(df))
  }

  f1 <- tempfile(fileext = ".rtf")
  f2 <- tempfile(fileext = ".rtf")
  generate_rtfreport(make_report(mtcars[1:3, ], "Table 1"), f1, overwrite = TRUE)
  generate_rtfreport(make_report(mtcars[4:6, ], "Table 2"), f2, overwrite = TRUE)

  out <- tempfile(fileext = ".rtf")
  assemble_rtf(c(f1, f2), out, overwrite = TRUE)
} # }
```
