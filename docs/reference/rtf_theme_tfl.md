# Clinical TFL preset (R6 theme)

R6 equivalent of \[rtf_table_style_tfl()\]. Returns a freshly
constructed \`rtf_theme\` matching the standard clinical TFL preset;
mutate its fields to broadcast the change to every referencing table.

## Usage

``` r
rtf_theme_tfl()
```

## Value

An \`rtf_theme\` (R6) object.
